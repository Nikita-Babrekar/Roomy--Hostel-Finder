# 🏠 Roomy | Aesthetic Student Hostel Discovery Platform

**Roomy** is a specialized web application designed to solve the housing crisis for university students. By focusing on high-quality UI/UX and verified listings, Roomy provides a seamless bridge between students and accommodation providers.

---

## 📖 Project Overview

### The Problem
Students moving to new cities often struggle with:
- **Scattered Information:** Looking through multiple websites and physical posters.
- **Lack of Transparency:** Not knowing the exact amenities or room conditions.
- **Safety Concerns:** Difficulty finding verified, secure PGs/Hostels near campus.

### Our Solution
**Roomy** offers a centralized, "aesthetic-first" platform where students can:
1. **Discover:** Browse curated listings with high-quality images.
2. **Filter:** Search based on budget, occupancy (Single/Shared), and proximity.
3. **Map View:** Visualize exactly how far a hostel is from their college.
4. **Direct Booking:** Streamlined contact and booking flow.

---

## 🛠️ Technical Architecture

The project follows a **Model-View-Controller (MVC)** pattern using the Flask framework.

| Component | Technology | Role |
| :--- | :--- | :--- |
| **Frontend** | HTML5, CSS3, JS | Aesthetic UI & Responsive Layout |
| **Framework** | Bootstrap 5 | Mobile-first grid and components |
| **Backend** | Python / Flask | Routing, Logic, and API handling |
| **Database** | SQLite | Storing User, Hostel, and Booking data |
| **Templating** | Jinja2 | Dynamic data rendering |
